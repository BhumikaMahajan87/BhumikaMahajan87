## Featured Projects

### [DocuMind — RAG-based Document Q&A](https://github.com/BhumikaMahajan87/documind)

A Retrieval-Augmented Generation (RAG) system that lets you chat with your documents. I built the full pipeline from scratch — document ingestion, sentence-aware chunking, embeddings, cosine-similarity vector search, and grounded answer generation with source citations. Features pluggable AI backends (OpenAI / Gemini / a fully-offline mode), a FastAPI REST API and web UI, 16 automated tests, Docker, and CI.

### [ShortLink — Scalable URL Shortener with Analytics](https://github.com/BhumikaMahajan87/shortlink)

A production-style URL shortener (Bit.ly–like) showcasing core backend and system-design concepts: Base62 short codes, LRU caching on the redirect path, token-bucket rate limiting, idempotent shortening, click analytics, and a pluggable storage layer (in-memory + SQLite). Built with FastAPI, with 27 tests run against both storage backends, Docker, and CI.

### [News Headline Classifier](https://github.com/BhumikaMahajan87/news-headline-classifier)

BERT-based news classification project with optimized preprocessing and low-latency inference.

### [Audio CAPTCHA Solver](https://github.com/BhumikaMahajan87/audio-captcha-solver)

Audio decoding system using MFCC/PCA features, classical ML models, and Conv1D CNN.

### [Image Captioning Model](https://github.com/BhumikaMahajan87/image-captioning-model)

CNN+LSTM image captioning pipeline with BLEU-based evaluation.

### [Emotion-Based Media Recommender](https://github.com/BhumikaMahajan87/emotion-based-media-recommender)

Component-based frontend for emotion-driven music and movie recommendations.

### Multimodal Image, Caption, Story & Audio Generator

Image-to-caption-to-story-to-audio pipeline using BLIP, FLAN-T5, and gTTS.
